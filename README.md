# MySoothe

<img width="1586" height="820" alt="image" src="https://github.com/user-attachments/assets/7a9b25da-8abd-4aaf-8f09-5488de430ad8" />

En este segundo ejercicio cuenta con diferentes apartados para tratar un mayor número de posibilidades ofrecidas por la aplicación. Entre ellas, he aprendido el uso de los tamaños para poder modificar los textos para que tengan el tamaño adecuado, lo mismo pasa con las imagenes donde explican el como usar diferentes imagenes para mostrar en la app y elcomo hacer que varias imagenes se muestren de forma consecutiva para, por ejemplo y en un caso de una app relacionada con redes sociales, mostrar diferentes fotos de los usuarios.

Además, tambien muestran formas para realizar los botones de navegación dentro de la aplicación y también la barra de búsqueda, son formas sencillas pero que vienen muy bien para ir aprendiendo a como hacerlas y tenerlas en cuenta a la hora de hacer la aplicación final.
